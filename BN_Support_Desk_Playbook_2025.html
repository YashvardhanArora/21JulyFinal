🏗️ System Overview

What is BN Support Desk?

The BN Support Desk is a comprehensive complaint management dashboard built specifically for BN Group India. It provides a centralized, secure platform for tracking, managing, and analyzing customer complaints with real-time updates and detailed analytics across multiple regions.

🌟 Key Features

🔐 Dual Access System

Separate Admin and ASM interfaces with role-based permissions for enhanced security and controlled access.

📍 Regional Mapping

Interactive Leaflet maps for Mathura (UP), Agra (UP), Bhimasar (Gujarat) with geographic complaint distribution visualization.

📊 Real-time Analytics

Live Recharts visualizations with automatic 30-second refresh for instant insights and trending analysis.

📤 Data Export

Excel export with 30-column business template structure for comprehensive reporting to stakeholders.

🔄 Status Tracking

Complete complaint lifecycle: New → Processing → Resolved → Closed with end-to-end visibility.

🔔 Real-time Notifications

WebSocket-based instant notifications with sound alerts for immediate awareness of new complaints.

📤 Data Export

📊 Excel Export Process

📄 Export Specifications

  • 📋 30-column business template structure
  • 📅 Date range: Full 2024 or 2025 data
  • 📊 Complete metadata inclusion
  • 📍 Geographic coordinates included
  • 💾 File format: .xlsx (Excel 2016+)
  • 📏 Empty fields marked with "-" for consistency

Export Workflow Steps

  1. Navigate to "All Complaints" section
  2. Select desired year (2024/2025)
  3. Apply any additional filters if needed
  4. Click "Export All Complaints" button
  5. Wait for data processing (2-5 seconds)
  6. Automatic download begins (.xlsx file)
  7. File saved to browser's default download folder

🔧 Troubleshooting

🚨 Common Issues & Solutions

❌ Login Problems

Admin login fails: Verify credentials (temp/temp), check caps lock, browser cookies enabled. Try clearing browser cache.

ASM login not working: Use correct credentials (asm/123 or demo/demo), username exactly as specified (no email domain). Try different browser or incognito mode.

⚡ Performance Issues

Slow dashboard loading: Check internet connection stability, browser performance tab for bottlenecks. Try refreshing page or restarting browser.

Charts not displaying: Enable JavaScript in browser, check console for error messages. Try updating browser to latest version.

📊 Data Issues

Complaints not showing: Verify year toggle selection (2024/2025), check filter settings aren't too restrictive. Try resetting filters to default.

Export failing: Disable popup blockers, check browser allows file downloads. Try right-click export button → "Save link as".
